# Semantic Drift Boundary Observer

Lightweight reference implementation for detecting semantic drift, response divergence, and boundary-level variation across text outputs.

## Features

- Compares two text inputs
- Estimates similarity and drift
- Labels drift severity
- Supports quick comparison of prompt or response variants

## Use cases

- Prompt iteration review
- LLM response comparison
- Output consistency checks
- Structured text-difference analysis

## Notes

This project uses lightweight text comparison heuristics for fast operational analysis.
